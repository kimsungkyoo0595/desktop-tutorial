이것은 Tailwind CSS로 만든 간단한 페이지입니다.

저희에 대해

이 페이지는 **HTML과 CSS**만을 사용하여 만들어졌습니다. **Tailwind CSS**를 활용하여 빠르고 쉽게 아름다운 디자인을 구현할 수 있습니다. 코드 내에서 직접 스타일을 적용하여 유지보수가 간편합니다.

주요 기능

  • 완벽한 **반응형 디자인**: 모든 기기에서 최적화된 화면을 제공합니다.
  • **클래스 기반 스타일링**: 직관적이고 효율적인 스타일링 방식입니다.
  • **가볍고 빠름**: 불필요한 코드가 없어 로딩 속도가 빠릅니다.
